## Overview

Full-stack web platform for Mesa RP - featuring real-time server integration, staff management, and community tools.

### Features

- **Live Server Stats** - Real-time player counts and server status
- **Staff Panel** - Discord-authenticated admin dashboard
- **Interactive Map** - Live player tracking with Leaflet
- **Community Hub** - Forums, events, and organization management
- **Character System** - Full character creation and management

---

## Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | Next.js 16 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS 4 + shadcn/ui |
| Database | MySQL + Prisma ORM |
| Auth | NextAuth.js + Discord OAuth |
| Realtime | Socket.io |

```bash

# Run development server
npm run dev

# Build for production
npm run build
```
